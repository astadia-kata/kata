Combining Zip Codes

In this package, you will find the stub code and tests for an Array Search exercise. There is no 'correct' answer but, an implementation is not considered complete until the unit tests for that class pass.

To implement, simply replace the thrown exception: throw new UnsupportedOperationException("implement me"); with your implementation. You can run tests by simply right clicking the corresponding test class and selecting run as->junit.

I strongly encourage writing more test cases to work out your algorithm. Also, as you learn to work with the standard API write tests that help to understand how the API works (instead of system.out.printlntt>). These tests can serve as 'notes' later on. Keep any new tests in your package.

Background

In this exercise, the scenario is, you are creating a site to allow querying for broadband providers based on a user's zip code.

You implementation will be passed sets of data containing a provider name and the zip codes supported by that provider. It should then be able to return an array of providers that support a given zip code.

The files can be found in the com.idea.exercise.zipcodes package.
