Combining Zip Code Data

In this exercise, the scenario is, you are creating a site to allow querying for broadband providers based on a user's zip code.

You implementation will be passed sets of data containing a provider name and the zip codes supported by that provider. It should then be able to return an array of providers that support a given zip code.

The files can be found in the 

com.idea.exercise.zipcodes
package.

Implementations

There are many ways to implement this. Try creating implementations:

  • with the most readable code
  • that use the least memory
  • that run the fastest

Be able to discuss your implementation.
